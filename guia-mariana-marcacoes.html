Como receber marcações no teu site

Comparação simples das ferramentas disponíveis — para escolheres sem dúvidas.

O que é isto?

Em vez de os pais te enviarem DM no Instagram para marcar, podem entrar no teu site, escolher o dia e hora que querem, e tu recebes a marcação automaticamente no telemóvel. Menos mensagens para trás e para a frente, mais tempo para ti.

🏆 A nossa recomendação
Cal.com
cal.com · Gratuito para sempre (plano individual)
💶 €0 / mês — grátis sem limites de tempo

É a ferramenta mais moderna do mercado em 2026. O plano gratuito tem tudo o que precisas: vários tipos de serviço, confirmações automáticas por email, sincronização com o Google Calendar e um widget que se encaixa diretamente no teu site. Não precisas de pagar nada.

Porque escolhemos esta?

Calendly foi durante anos a mais popular, mas agora está desatualizada e cara. Cal.com oferece mais funcionalidades no plano grátis do que qualquer alternativa, incluindo múltiplos tipos de serviço (Noturno, Semana, FDS) — o que o Calendly gratuito não permite.

🚀 Como configurar o Cal.com — passo a passo
Tempo estimado: 20–30 minutos

Faz isto uma vez, e depois é automático. Só tens de gerir as tuas marcações no telemóvel.

1
Criar conta gratuita

Vai a cal.com → clica em "Get started for free" → regista com o teu email (marianamarques300802@gmail.com).

2
Ligar o Google Calendar

Em Configurações → Calendários → Adicionar Google Calendar. Assim a Mariana não recebe marcações quando já está ocupada.

3
Criar os 3 tipos de serviço

Em "Event Types" → New Event Type. Criar um para cada serviço:

• Babysitting Noturno — €15/h — disponível noites e fins de semana
• Após as 19h30 (Semana) — €10/h — seg a sex após 19h30
• Fim de Semana — €12/h — sábados e domingos

4
Configurar as horas disponíveis

Em "Availability" define os teus horários reais. Para cada serviço podes ter uma disponibilidade diferente.

5
Adicionar perguntas ao formulário de marcação

Em cada Event Type → "Questions" → adicionar:
• "Idade e nome da(s) criança(s)"
• "Morada / localidade"
• "Alguma informação importante sobre a criança?"

6
Copiar o código do widget

Em cada Event Type → "Embed" → copia o código HTML. Depois envias ao Telmo para ele inserir no teu site. É uma operação rápida.

7
Instalar a app no telemóvel

Descarrega a app "Cal.com" na App Store ou Google Play. Aí recebes notificações de novas marcações, podes confirmar ou cancelar, e ver o teu calendário.

🌐 Domínio para o teu site
O que é um domínio?

É o endereço do teu site. Em vez de um link longo como "telmodfs.github.io/mariana-babysitting", passaria a ser algo como "marianababysitting.pt" — mais profissional e fácil de memorizar.

Sugestões de nome:

DomínioExtensãoPreço estimadoNota
marianababysitting.pt.pt~€10–15/anoIdeal — local, profissional
babysitting-mariana.pt.pt~€10–15/anoBoa alternativa
marianababysitting.com.com~€10–12/anoOK, menos local
Onde registar?

Em Portugal, os melhores registradores são dominios.pt, lusoaloja.pt ou site.pt. O registo .pt custa entre €10–15 por ano. Podes verificar se o nome está disponível nessas páginas antes de decidir.

⚠️ Importante

O GitHub Pages (onde o site está agora alojado) permite ligar um domínio próprio gratuitamente. Isso significa que o custo anual é apenas o do domínio — sem alojamento adicional. O Telmo pode configurar isto para ti em menos de 30 minutos.
