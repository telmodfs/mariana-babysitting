🔧 Como ligar o domínio ao GitHub Pages (Volvara)

1. Comprar domínio em dominios.pt ou lusoaloja.pt
2. Nas definições DNS do registrador, adicionar CNAME: www → telmodfs.github.io
3. Adicionar A records apontando para IPs do GitHub Pages (185.199.108–111.153)
4. No repositório GitHub → Settings → Pages → Custom domain → inserir domínio
5. Ativar "Enforce HTTPS" — certificado SSL gratuito automático
⏱️ Propagação: 24–48h
